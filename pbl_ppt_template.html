RAही

The Traveler's Social Network

"A full-stack, cross-platform mobile application built exclusively for Indian travelers. Seamlessly blending geographic milestone tracking, real-time social networking, and AI-powered trip planning into a single ecosystem."

Project Abstract

Developed as a B.Tech minor project (PBL-4) using React Native, Expo, TypeScript, and Firebase. RAही's core distinguishing feature is its offline-first smart check-in system, utilizing the Turf.js geospatial library and bundled GeoJSON datasets to automatically identify visited Indian states and Union Territories via GPS.

Smart Tracking

Auto-detect Indian States via GPS & Turf.js

Real-Time Sync

Zero-latency updates via Cloud Firestore

AI Integration

Smart itineraries powered by LLaMA 3

Problem Statement

  • ▸App Fragmentation: Current apps focus on single aspects — booking, navigation, or reviews. No unified travel network exists.
  • ▸No Geographic Gamification: Travelers lack an automated way to track and visualize their journey history across Indian states.
  • ▸Missing Privacy Layers: Existing travel tools lack request-gated social ecosystems ensuring location maps are shared only with trusted connections.
  • ▸Manual Planning: Creating structured, budget-aware itineraries requires extensive manual research across multiple platforms.

Scope of RAही

  • ▸ Cross-platform deployment (iOS/Android) via 100% TypeScript codebase
  • ▸ Offline geospatial point-in-polygon computation
  • ▸ Hybrid content feed with adaptive layout rendering
  • ▸ Request-gated follower model & privacy enforcement
  • ▸ Interactive personal travel map with dynamic bounding
  • ▸ On-demand AI itinerary generation via Groq API

Functional Modules

Smart GPS Check-In

  • ▸ High-accuracy coordinate acquisition via expo-location
  • ▸ Offline GeoJSON processing using Turf.js
  • ▸ Instant Indian state/UT identification
  • ▸ Gamified state-counter on user profiles

Hybrid Social Feed

  • ▸ Adaptive UI rendering for Photo vs. Text posts
  • ▸ Real-time data push via Firestore onSnapshot listeners
  • ▸ Infinite scroll pagination & atomic write batches
  • ▸ Follower-request gated privacy ecosystem

Interactive Travel Map

  • ▸ Plots all GPS-tagged posts as custom markers
  • ▸ Native map rendering via react-native-maps
  • ▸ Dynamic camera bounding based on post extents
  • ▸ Exclusive visibility to approved followers

AI Trip Planner

  • ▸ LLaMA 3 integration via Groq Cloud API
  • ▸ Dynamic prompt construction (Destination, Duration, Budget)
  • ▸ Generates structured day-wise itineraries
  • ▸ Includes accommodation, food, and cost estimations

Market Positioning

The travel tech market is polarized between Utility (Booking/Navigation) and Content (Blogs/Reviews). RAही identifies a "Blue Ocean" in the middle: Social Gamification specifically tailored for the Indian subcontinent.

Feature Coverage Comparison

The Gamification Pivot

While competitors track sheer distance, RAही tracks Coverage (States/UTs) via Turf.js. Collecting Indian States serves as a highly localized, culturally relevant milestone mechanism.

Micro-Blogging vs Long Form

Platforms like Tripoto require high-effort blogging. RAही leverages an Instagram-style feed, lowering the friction: Snap, Check-in, Post.

Integrated Trip Planning

Instead of forcing users to exit the app to research via ChatGPT or Google, RAही includes an in-house LLaMA 3 Planner to generate instant, context-aware itineraries.

Social Privacy Architecture

Pure trackers act as public links. RAही operates as a Social Network first, requiring target-user approval before travel maps and GPS locations are exposed.

Strategic Outcome

RAही successfully integrates distributed backend systems, client-side geospatial processing, and modern cross-platform UI to deliver a production-ready application for the Indian travel community.

Conclusion & Future Work

RAही demonstrates the practical integration of modern mobile development technologies. By combining React Native, Firebase, Turf.js, and LLaMA 3, the project achieves its goal of creating a comprehensive social ecosystem for travelers.

Future Roadmap: Imminent enhancements include FCM Push Notifications, real-time Direct Messaging, offline post drafting, EAS Build deployment to app stores, and full-text search integration via Algolia.
